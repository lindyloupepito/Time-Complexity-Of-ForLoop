Time-Complexity-Of-ForLoop
==========================
Displays time complexity for simple for loops like

for(int i=0; i<n; i++){
    sum = i;
    sum = sum + 2;
    sum += 9;
}

for(int i=2; i<=n; i+=3){
    sum = i;
    sum = sum + 2;
    sum += 9;
}

for(int i=n; i>=0; i--){
    sum = i;
    sum = sum + 2;
    sum += 9;
}

for(int i=n; i>0; i-=3){
    sum = i;
    sum = sum + 2;
    sum += 9;
}

for(int i=n; i>0; i/=3){
    sum = i;
    sum = sum + 2;
    sum += 9;
}

for(int i=1; i<n-3; i*=2){
    sum = i;
    sum = sum + 2;
    sum += 9;
}
